# Cows on Skates (6191)

[Cows on Skates](https://www.acmicpc.net/problem/6191)

## 문제 요약

격자 모양의 농장에서 장애물인 바위를 피해 (1, 1) 지점에서 (R, C) 지점까지 가는 경로를 찾는다.
이동은 상하좌우 인접한 칸으로 이동 가능하다.
찾아낸 경로 중 한가지를 순서대로 출력한다.
경로는 최소 1개 이상을 존재함을 보장한다.

## 문제 풀이 접근

시작점에서 도착점까지 경로를 보장하기 때문에 bfs 방식으로 그래프 탐색을 진행한다.
BFS 탐색 과정에서 각 격자에 도달할 수 있는 최단 거리를 기록한다.
이 과정을 진행하면 도착점(R, C) 격자에는 최단 거리 값이 기록된다.
도착점(R, C)부터 시작하여, 현재 격자의 최단 거리 값(visited)보다 1 더 작은 위치로 이동한다.
이러한 반복을 수행하면 시작점(1,1)에 도달할 수 있다.
